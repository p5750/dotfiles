# LazyGit Configuration

Location: `config/lazygit/config.yml`

## Settings

- `disableStartupPopups`: true
- `quitOnTopLevelReturn`: true - pressing q at top level quits
- `promptToReturnFromSubprocess`: false
- `notARepository`: quit - exits if not in a git repo
- `gui.showIcons`: true

## Neovim Integration

Opened from Neovim via `<leader>lg` (configured in `config/nvim/lua/plugins/lazygit.lua`).
